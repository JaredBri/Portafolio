Portafolio Profesional - Desarrollo web Front End

Este proyecto consiste en la creación de un portafolio personal tipo Single Page, desarrollado con HTML5 y CSS3.
su objetivo es presentar mi perfil profesional, habilidades y proyectos, aplicando buenas practicas de desarrollo web y diseño responsivo.

El sitio incluye secciones como:
- Presentación
- Sobre Mi
- Habilidades
- Proyectos
- Formulario de contactos
- Footer con redes sociales

Tecnologias Usadas:
- HTML5 (Estructura semantica)
- CSS3 (Estilos, Flexbox y Grid)
- Git y GitHub (Control de excepciones)

Sus funciones principales son:
- Navegación interna mediante anclas
- Diseño moderno y organizado
- Uso de Flexbox y Grid
- Tabla de habilidades con barras de progreso
- Formulario de contacto con validación HTML5
- Footer con enlaces a redes sociales
- Diseño adaptable (responsive)
